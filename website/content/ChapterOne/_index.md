# 函数使用场景     
shell里的函数可以用来将一串操作代码封装，特别是这串代码具有重复性时可以考虑使用函数  

# 函数写法      
三种   
**第一种**   
```bash
function function_name {
command...
}
```   

**第二种**   
```bash
function_name () {
command... 
}
```   

**第三种**
```bash
funtion_name ()
{
command...
}
```
